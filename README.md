## 🧑‍💻 About Me

I run the IT infrastructure for a German manufacturing company by day - Active Directory, Exchange, Veeam, the usual sysadmin stack - and stream / create content on the side as a registered sole proprietor.

I'm **not** a professional developer. I keep finding gaps where the "right tool" either doesn't exist, has a terrible UI, or costs €4k/year per seat. So I build my own. The repos here are a mix of:

- 🛠️ **Sysadmin automation** - PowerShell, mostly. AD, Exchange, certificates, reporting, the kind of stuff that should already exist but somehow doesn't.
- 🎮 **Streaming & content tools** - OBS pipelines, Twitch/Discord bots, overlay widgets, post-stream automation.
- 🏠 **Homelab tinkering** - Proxmox, TrueNAS, self-hosted everything I can get my hands on.
- 📚 **Learning projects** - Currently working my way through C# / .NET 10 properly.

If something here is useful to you - go for it and use it.

## ⭐ Featured Project

### 🎬 [MortysDLP](https://github.com/MortysTerminal/MortysDLP)

A user-friendly Windows app for downloading videos and audio from various platforms. Built in C# because the existing solutions either had clunky UIs or required command-line knowledge most people don't have.

> *"Quick clip downloads from anywhere, no broken builds, no shady websites — a tool that opens, works, and stays current."*
